## demo 

![demo gif](https://raw.githubusercontent.com/quadnix/gameoflife/master/demo.gif)

## compiling

install [sfml](http://sfml-dev.org/), then `make`.

## keys

key | result
----|-------
`space` | step
`r` | randomize data
`c` | enable continuous mode

when not in continuous mode, click to toggle blocks
